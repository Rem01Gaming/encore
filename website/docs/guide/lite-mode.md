# Lite Mode

Encore Tweaks can run very hot and power hungry when running game, especially on modern devices. Lite mode tries to "nerf" the tweaks to reduce power usage and heat. Lite Mode can be enabled from game settings individually per game or enabled from settings page globally.

## When to Use Lite Mode?

- **Sustained Gaming**: For long sessions where you want to avoid the device becoming uncomfortable to hold.

- **Battery Longevity**: When you want a smoother experience without the heavy drain of maximum performance.

- **Low-Demand Games**: For titles that don't need maximum performance of your device but still benefit from Encore Tweaks.
